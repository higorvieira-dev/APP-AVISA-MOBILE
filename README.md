# APPAVISA Mobile v5

Versão funcional com:
- Painel do atleta, professor e diretoria por cargo.
- Professor: aplicar advertência, solicitar material, solicitar evento e ver agenda publicada.
- Diretoria: horas zeradas, pautas, arquivos administrativos, convites, cupons de desconto e aprovação de eventos.
- Loja: desconto aplicado por cupom, valor final correto e voucher no histórico.
